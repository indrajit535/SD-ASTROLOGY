# Assets Directory

This directory contains all image assets for the AI Palm Reader app.

## Required Assets:

### 1. Background Images
- `splash-bg.jpg` - Splash screen background (1920x1080)
- `cosmic-bg.jpg` - Cosmic background for app (1920x1080)

### 2. App Logo
- `palm-logo.svg` - Palm reader logo (500x500, transparent)

### 3. Tarot Cards (22 cards)
- `tarot-cards/card-01.jpg` to `tarot-cards/card-22.jpg`
- Each card: 400x600 pixels
- Standard Rider-Waite tarot deck images

### 4. Zodiac Icons (12 icons)
- `zodiac-icons/aries.svg` to `zodiac-icons/pisces.svg`
- Each icon: 100x100 pixels, colored

### 5. Onboarding Illustrations (3 images)
- `onboarding-1.svg` - Tarot introduction
- `onboarding-2.svg` - Palm reading
- `onboarding-3.svg` - AI guidance
- Each: 300x300 pixels

## Asset Guidelines:
- All images should be optimized for mobile
- Use dark theme colors (blues, purples, violets)
- Maintain consistent style across all assets
- Ensure SVG files are properly formatted
- Compress JPG files for faster loading

## Alternative: Use Placeholders
For development, you can use placeholder services or create simple colored divs.
